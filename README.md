# What
4kb PC intro released at TRSAC 2025

You will need minimal_gl to build. Load the project file balance.json in minimal_gl and export the executable from the menu.

# Using
* minimal gl: https://github.com/yosshin4004/minimal_gl/ version 2025_05_28
    * There is a bug in this version, you may need to enable render targets to export executable
* Crinkler: https://github.com/runestubbe/Crinkler version 2.3
* Shader Minifier: https://github.com/laurentlb/shader-minifier/ version 1.5.1
